Task 2 – Stock Price Prediction

Objective:
Predict the closing price of a stock based on historical features such as opening price, high, low, and volume.

Dataset:
stock_data.csv (manually created for this task)

Features Used:
- Open
- High
- Low
- Volume

Target:
- Close

Model Used:
- Linear Regression (from scikit-learn)

Evaluation Metrics:
- Mean Squared Error (MSE)
- R² Score
- Line Plot (Actual vs Predicted Closing Prices)

Instructions:
1. Make sure `stock_data.csv` is in the same folder as the script.
2. Run `stock_prediction.py` using Python 3.
3. Required libraries: pandas, matplotlib, scikit-learn.
